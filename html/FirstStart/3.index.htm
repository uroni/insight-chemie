Die Moleküldatenbank ergänzen

Die Moleküldatenbank von Chemator kann beliebig ergänzt werden. Die einfachste Methode neue Moleküle der Datenbank hinzuzufügen wird nun hier beschrieben.

Begeben Sie sich in das Wurzelverzeichnis von Chemator (Standardmäßig C:\Programme\Chemator). Dort befindet sich ein Verzeichnis namens "Scripts". In diesem Verzeichnis befindet sich die Komplette Datenbank geordnet als Dateien. Sie könne auch erkennen, dass die diesem Verzeichnis untergeordneten Ordner jeweils der Baumansicht in Chemator entsprechen.
Um nun ein neues Molekül hinzuzufügen müssen Sie es erst in Chemator konstruieren. Verwenden Sie hierzu bereits existierende Moleküle oder die Entsprechenden Atome. Falls sie nicht mit der veränderung von Molekülen vertraut sind können Sie es hier nachlesen: Ein Molekül bearbeiten.
Nun können sie das neu erstellte Molekül mit [Datei->Speichern] in den gewünschten Unterordner von "Scripts" abspeichern um es der Moleküldatenbank hinzuzufügen. Nach einem Neustart von Chemator ist das neu erzeuge Molekül auswählbar und erzeugbar. Möchten Sie, dass das Molekül einen speziellen Namen besitzt müssen Sie es vor dem abspeichern mit "Name ändern" umbenennen.
