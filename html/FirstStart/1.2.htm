Auswahl und Erzeugung von Molekülen

Rechts oben sehen Sie eine sogenannte Baumansicht. Dort werden Sie, untergliedert in verschiedene Bereiche, dreidimensionale Molekülmodelle finde. Um ein Molekül dem linken Arbeitsbereich hinzuzufügen könne Sie auf Moleküle auswählen und dann sich solange durch die Kategorien klicken, bis Sie das gewünschte Molekül erreicht haben. Markieren Sie das gewünschte Molekül mit einem einfachen Linksklick und wählen sie entweder "erzeugen", oder Doppelklicken Sie auf den Eintrag.
Moleküle können auch aus dem Kontext einer erklärenden Seite, wie dieser hier, erzeugt werden. Dazu einfach auf den entsprechenden Link klicken und das Molekül wird automatisch ihrem Arbeitsbereich hinzugefügt: Ein Wassermolekül erzeugen
